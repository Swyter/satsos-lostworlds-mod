To enable the "Dev" mode, first you need to launch the game with the parameter "-dev", here is a guide for the Steam version of the game, but these parameters are also valid for the Gog version of the game.

1 - First we need to go to steam and find the game, then click on "Properties":

2 - Click on Launch options...

3 - For this case, we will need to use two parameters:

  • The first one is just to enable the dev mode, we only have to write -dev.
  • The second one is used to launch the mod. The format is -mod "PathToFolder", could be relative or absolute. Is also very important that the path to the folder uses the quotation marks to avoid problems.

-dev -mod "../../workshop/content/606710/1614156774"

4 - For the case that you want to modify the inventory or the game objectives, you also need to create a folder called Albert in you game installation directory. Inside this folder, we have to put the Hashcodes.h, in the case that you does not have the Authoring Tools DLC installed, you can get the file from this link.
